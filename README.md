# Mi Sitio Personal

## Descripción

Este proyecto corresponde al desarrollo de un sitio web personal realizado como práctica del módulo de HTML5.

El objetivo es aplicar la estructura semántica de HTML utilizando las principales etiquetas semánticas, una correcta jerarquía de encabezados y navegación entre diferentes páginas.

## Estructura del proyecto

```
mi-sitio-personal/
│
├── index.html
├── pages/
│   ├── sobre-mi.html
│   ├── proyectos.html
│   ├── habilidades.html
│   └── contacto.html
├── img/
└── README.md
```

## Tecnologías utilizadas

- HTML5

## Características

- Estructura semántica con:
  - `<header>`
  - `<nav>`
  - `<main>`
  - `<section>`
  - `<article>`
  - `<figure>`
  - `<figcaption>`
  - `<footer>`
- Navegación entre las diferentes páginas.
- Imágenes con atributo `alt` para mejorar la accesibilidad.
- Organización del proyecto mediante carpetas.

## Páginas

- Inicio
- Sobre mí
- Proyectos
- Habilidades
- Contacto
